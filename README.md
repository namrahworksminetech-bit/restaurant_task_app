# Restaurant Task App

A modern Flutter restaurant browsing application with clean UI, responsive layouts, pagination, retry handling, splash screen, and custom app icon support.


## Environment Specs

 Flutter SDK: 3.x.x
 Dart SDK: 3.x.x
 Gradle: 8.x
 Android Gradle Plugin: dev.flutter.flutter-gradle-plugin
 Kotlin JVM Target: 17
 Java Source Compatibility: Java 17
 Java Target Compatibility: Java 17
 JDK: 17
 Application ID: com.example.restaurant_task
 Min SDK: flutter.minSdkVersion
 Target SDK: flutter.targetSdkVersion
 Compile SDK: flutter.compileSdkVersion
 NDK Version: flutter.ndkVersion
 Build Variant: release

## Features

 Restaurant listing screen
 Restaurant detail screen
 Infinite scrolling / pagination
 Retry handling for API failures
 Responsive layouts
 Splash screen
 Custom app icon
 Expandable descriptions
 Material 3 UI


## Getting Started

Install dependencies:

```bash
flutter pub get
```

Run the app:

flutter run


Build release APK:

flutter build apk --release
